Posted on 2019-10-10 | In python

How does objective be passed in Python?

这几天学习Python和帮同学解决问题中遇到了一个神奇的小问题。

如果指定一个list为a，运行b=a，对b做更改，a的值也会同样更改。这引起了我深入了解python中值的传递的深层原理。

python中一切皆对象，这句话刚开始学习的时候并没有任何感觉。

但随着深入了解编程，python的确大不同于C或者java。所以这里很难说一个整数、浮点数或者一个list是value或者reference。所以这里用objective，记录我学到的Python是如何传递和引用对象的。

事实上，python中有两种objective: mutable & immutable

  1. mutable包含list, set, dict

  2. immutable包含bool, int, float, tuple, str, frozenset

MNIST

Posted on 2019-10-04 | In Machine Learning , dataset

MNIST 数据集来自美国国家标准与技术研究所, National Institute of Standards and Technology (NIST). 训练集 (training set) 由来自 250 个不同人手写的数字构成, 其中 50% 是高中学生, 50% 来自人口普查局 (the Census Bureau) 的工作人员. 测试集(test set) 也是同样比例的手写数字数据.

MNIST 数据集下载地址： http://yann.lecun.com/exdb/mnist/

包含如下四个部分：

  • Training set images: train-images-idx3-ubyte.gz (9.9 MB, 解压后 47 MB, 包含 60,000 个样本)
  • Training set labels: train-labels-idx1-ubyte.gz (29 KB, 解压后 60 KB, 包含 60,000 个标签)
  • Test set images: t10k-images-idx3-ubyte.gz (1.6 MB, 解压后 7.8 MB, 包含 10,000 个样本)
  • Test set labels: t10k-labels-idx1-ubyte.gz (5KB, 解压后 10 KB, 包含 10,000 个标签)

Next theme

Posted on 2019-10-03 | In Blog

如何美化NEXT主题

1. 添加动态背景

NEXT主题5.1.1版本以上内置该功能(太受欢迎)。

配置文件 themes/next/_config.yml中找到：

改为：

canvas_nest: true

想要更改该背景的参数：

文件所在位置themes\next\source\lib\canvas-nest\canvas-nest.min.js

参数说明：

  • color: 线条色彩，默认"0,0,0" 三个值分别对应（R,G,B)
  • opacity: 线条透明度，默认 0.5，取值范围 0~1
  • count: 线条总数，默认99
  • zIndex 该背景的z-index属性，CSS属性用于控制所在层的位置，默认-1
